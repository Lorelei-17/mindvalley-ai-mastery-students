## Troubleshooting Help

If I'm stuck during setup or onboarding, help me troubleshoot:

### Common Setup Issues

**N8N Problems:**
- "Workflow won't import" → Check I have Starter plan or higher
- "Node not found" → May need community nodes (Settings → Community Nodes)
- "Can't find Execute button" → It's at the **bottom center** of the canvas (red button)
- "Don't see output" → Click Executions tab → click the run → click the node

**API Key Problems:**
- "Invalid API key" → Check for extra spaces when copying
- "403 Forbidden" → Key may be from wrong service or expired
- "Model not found" → Try `gemini-2.0-flash` instead of older model names

**Claude Code Problems:**
- "Extension not responding" → Restart VS Code
- "Can't sign in" → Check Claude Pro subscription is active
- "Permission denied" → Run `chmod +x setup.sh` for scripts

### How to Get Help

1. **Describe the issue** - What step are you on? What error do you see?
2. **Share screenshots** - I can't see your screen, so screenshots help a lot
3. **Tell me what you tried** - What have you already attempted?

If I can't solve it, I'll tell you to:
- Post in the course Slack channel with screenshots
- Tag Tyler for urgent issues
- Wait for live session troubleshooting
